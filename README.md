## Deskripsi Aplikasi
1. Counter Sederhana
   Halaman ini berisi fitur penghitung angka yang bisa dinaikkan atau diturunkan dengan tombol
   Tombol (+) untuk menambah angka
   Tombol (-) untuk mengurangi angka
   Sehingga tampilan angka akan berubah secara real time
   
3. Memahami Data dari API
   Halaman ini digunakan untuk menampilkan data yang diambil dari API, untuk aplikasi yang kami buat kali ini menampilkan informasi terkait Universitas Pancasila. Informasi yang kami ambil dari internet

4. Profil Anggota Kelompok 8
   Halaman ini berisi informasi mengenai anggota kelompok yang membuat aplikasi, tampilan yang kami buat kali ini dalam bentuk card, yang berisi foto, nama, tempat tanggal lahir dan alamat atau tempat tinggal
